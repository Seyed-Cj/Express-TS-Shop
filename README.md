# Shop Api

shop api with ts and express

## Usage

```bash
git clone https://github.com/Seyed-Cj/Express-Typescript
cd Express-Typescript
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Environment

Create `.env` and define variables like:

```
PORT=3000
NODE_ENV=development
JWT_SECRET=your secret
JWT_REFRESH_SECRET=your secret
```

## Purpose

A small, ready-to-extend base for building APIs with Express + TypeScript.
